Request module¶

In this module, both subclasses IndividualRequest and GroupRequest override the abstract methods in the super class Request, which facilitate the interaction between group and students.

class Request.GroupRequest¶

Bases: Request

This class inherits the abstract class Request, handling the request operations between an individual (student) and a group.

id¶

the identifier of the request

Type

int

status¶

indicating the status of the request (whether it is active or not)

Type

int

sender¶

the user_id of the student who sent the request

Type

int

receiver¶

the group_id of the group who received the request

Type

int

accept_request()¶

This method operates by invoking the add_member method of a group to add the sender, which updates the relevant information.

decline_request()¶

This method operates by invoking the notification of the sender and nothing else happens.

class Request.Request¶

abstract accept_request()¶

This method operates to accept the request and update both sender and receiver.

Note

This method must be implemented by any subclass.

abstract decline_request()¶

This method operates to decline the request and notify the sender if necessary.

Note

This method must be implemented by any subclass.
