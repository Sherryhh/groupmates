Group module¶

Group Class

This class records the basic information of a group, keeps all the members belonging to this group and handles the requests related to this group.

class Group.Group¶

Bases: object

name¶

the name of the group

Type

str

members¶

a list recording the members in the group, should have length at least 1 and at most 6 (6 means the group is full)

Type

list(Student)

id¶

the unique identifier of the group among all the groups

Type

int

languages¶

a list recording all the languages that are used by the members inside the group

Type

list(str)

missing_skills¶

a brief paragraph stating which kinds of skills are missing in the current group

Type

str

group_request¶

a list recording all the group requests the group has received

Type

list(Request)

open¶

True if this group still has vacant seats (so it will show in the group list) By default, it is set True when a new group is created.

Type

bool

add_member(new_member_id)¶

This method enables to add a new member into the group. This operation should only be successful if the group size is lower than 6.

Parameters

new_member_id (Student) – the user_id of the new member of the group

Returns

(bool) True if the group is not full, and False otherwise.

change_group_name(new_name)¶

This method changes the name of this group. Notice that the new_name should not be identical to any existing group name, otherwise the modification will fail.

Returns

(bool) True if the modification succeeds, and False otherwise.

get_all_group_requests()¶

This method returns all the group requests the group received.

Returns

(list(json)) a list of group requests instances

static get_all_groups_info()¶

It returns all the group instances created before as a list of json where each entry in the list has the pre-defined structure.

Returns

list(json) a list of group instances

Note

This method is static for class Group.

get_all_members()¶

This method returns a list of all the members inside this group.

Returns

(list(Student)) a list of group members

get_group_info()¶

Return public and basic information of the group in a pre-defined manner.

Returns

(json) pre-defined structure of attributes in a group

set_missing_skills(skills)¶

This method sets the “missing skills” part of the group information. It can be a short paragraph indicating the desired skills that any student wanting to join the group may have.

Parameters

skills (str) – a brief paragraph of desired skills

Returns

(bool) True if the modification succeeds, and False otherwise.
