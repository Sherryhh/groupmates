sortIndividuals(allStudents)¶

This method takes the information of all the other students and compare their information with the (self) user, based on the languages they use, the frontend and backend frameworks they use. For each other student, this method gives a matching score. Finally, it returns a list of dictionaries where each dictionary represents the information of a student sorted by their matching scores with the (self) user decreasingly.

Parameters

allStudents (list(dict)) – the information of all the students.

Returns

a list of dictionaries where each dictionary represents the information of a student sorted by their matching scores with the (self) user decreasingly.

Return type

(list(dict))
